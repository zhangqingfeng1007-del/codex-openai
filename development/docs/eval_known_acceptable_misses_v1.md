## 明细

### 1. 产品 1803 × 交费频率 — mismatch（不是 miss）

| 项目 | 内容 |
|------|------|
| eval 表现 | mismatch（有值但不对） |
| 提取值 | `"趸交，年交"` |
| gold 值 | `"趸交，年交，半年交，季交，月交"` |
| 根因 | 提取值来自推断规则（从交费期间 `"趸交，5/10年交"` 推断），仅含趸交和年交；完整频率（含半年交、季交、月交）来源为**核保规则文件**，本地无此文件 |
| 建议处理 | 该 mismatch 为 `not_self_sufficient` 性质，计算命中率时可标注为非代码缺陷 |
| 是否影响交付 | 不影响。提取链路无 bug，缺的是外部文件 |

### 4. 产品 889 × 多字段 — miss（结构性，非代码 bug）

| 字段 | eval 表现 | 根因 |
|------|-----------|------|
| 保险期间 | miss | 条款原文为"由您与我们约定并在保险合同上载明"，无具体值 |
| 交费期间 | miss | 条款无"交费期间"字样（0 blocks），以"缴费期限"等替代，rate table 亦无此产品 |
| 宽限期 | miss | blocks 碎片化，"60日"与"宽限期"标题跨 block，当前规则未覆盖 |
| 重疾赔付次数 | miss | blocks 碎片化，规则未匹配 |
| 重疾分组 | miss | 候选值为空字符串，不命中 gold |
| 投保年龄 | miss | 同 1568，条款无标准年龄范围格式 |
| 交费频率 | **mismatch** | 仅提取 "月交"，gold 为 "趸交，年交，半年交，季交，月交"，完整频率来自核保规则 |
