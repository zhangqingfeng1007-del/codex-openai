## 一、格式分类

10款重疾费率文件分三种格式：

| 格式 | 缴费期位置 | 代表产品 | 文件类型 |
|------|-----------|---------|---------|
| **格式A（列头式）** | 缴费期值作为列名出现在表头行 | 851A, 1134, 864, 1464, 1568, 1578, 864, 919A | PDF |
| **格式B（行数据式）** | 缴费期值作为数据列中的值，每行一条记录 | 889 | PDF |
| **格式C（XLSX）** | 同格式A，但为电子表格 | 1803, 1568, 1578, 919A | XLSX |

## 二、各产品详细结构

### 889 — 格式B（行数据式 PDF）

**文件**：`889-中信保诚「惠康」重大疾病保险（至诚少儿版）-费率表.pdf`
**页数**：3页
**单位**：每1000元保险金额，人民币元

**表头（第1页，y坐标行）**：

| 行位置 | 内容 |
|--------|------|
| 第0行 | 产品名称（标题） |
| 第5-7行 | 说明文字（含交费频率换算公式） |
| 第9行 | `年龄` |
| 第10行 | `性别` |
| 第11行 | `缴费期` |
| 第12行 | `保障期` |
| 第13-30行 | 标准保费及各加费等级 |

**数据行结构**（每组4行 + N个保费行）：

```
年龄值（如 0）
性别值（男 / 女）
缴费期值（1年 / 5年 / 10年 / 15年 / 20年 / 25年 / 30年）
保障期值（终身）
标准保费
EM_25 加费
...
```

**缴费期完整枚举**（全文本中出现的所有值）：

```
1年  →（别名：趸交）
5年
10年
15年
20年
25年
30年
```

**注意**：1年（趸交）大量集中在前几百行，5/10/15年出现更晚。逐行扫描窗口过小（<150行）会漏掉后段缴费期。

**交费频率说明**（第1页第7行文字）：

```
本表为年缴的费率表，月缴保费=年缴保费×0.09，季缴保费=年缴保费×0.27，半年缴保费=年缴保费×0.52
```

→ 交费频率：年交 / 月交 / 季交 / 半年交（从该行正则提取）

## 三、缴费期别名归一化规则

所有费率表中出现的缴费期原始值 → 标准值：

| 原始值 | 标准值 | 来源产品 |
|--------|--------|---------|
| `1年` | `趸交` | 889 |
| `一次性付清` | `趸交` | 1803 |
| `趸交` | `趸交` | 851A |
| `5年交` / `5年` | `5年交` | 多个 |
| `10年交` / `10年` / `10` | `10年交` | 多个 |
| `15年交` / `15年` / `15` | `15年交` | 多个 |
| `20年交` / `20年` / `20` | `20年交` | 多个 |
| `25年交` / `25年` | `25年交` | 889 |
| `30年交` / `30年` / `30` | `30年交` | 多个 |
| `3年交` / `3年` | `3年交` | 1803 |

## 五、rate_table_extractor.py 开发建议

### 5.1 格式检测逻辑

```
XLSX：
  → 读取前 8 行所有非空单元格
  → 若某行出现 "趸交" / "一次性付清" / "N年交" → 确认为格式C，该行即列头行

PDF（格式A vs 格式B）：
  → 先尝试坐标读取（get_text('words')）
  → 若同一 y 坐标行出现 ≥3 个"N年" / "N年交" / "趸交" → 格式A
  → 否则 → 格式B（缴费期值出现在数据列中）
```

### 5.2 格式A（列头式 PDF）提取

```python
# 1. 找表头行：包含 ≥3 个年期关键词的 y 坐标行
# 2. 提取该行所有词作为列名
# 3. 规范化列名（补充"年交"后缀、别名替换）
# 4. 扫描后续所有行（不限窗口大小）
```

### 5.3 格式B（行数据式 PDF）提取

```python
# 1. 找"缴费期"列的位置（出现在某 y 坐标行的 x 坐标）
# 2. 扫描全文所有行，提取该 x 位置的值
# 3. 去重即为所有缴费期 → 规范化
# ⚠️ 禁止用固定行窗口（如 line[12:152]），必须全文扫描
```

### 5.4 频率提取

```python
# 1. 若 PDF：扫描前3页所有文字，正则匹配频率换算公式
# 2. 若 XLSX：同样扫描所有单元格文字
# 3. 若无：extracted_fields.pay_frequencies = []，下游从条款补填
```
